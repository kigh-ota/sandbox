Adjacent siblings

margin-bottom: 10px;
margin-top: 10px;

Parent and first child

親要素のmargin-topと子要素のmargin-topとを 隔てるborder, padding, インラインコンテンツなどが存在せず完全に重なり合うような場合、 marginが相殺して子要素のmarginが親要素を突き抜ける。
margin-top: 20px; (child)
margin-top: 0; height: unset;

Parent and last child

親要素のmargin-bottomと子要素のmargin-bottomとを 隔てるborder, padding, インラインコンテンツ, heightなどが存在せず完全に重なり合うような場合、 marginが相殺して子要素のmarginが親要素を突き抜ける。
margin-bottom: 0; height: unset;
margin-bottom: 20px; (child)
margin: 0;
